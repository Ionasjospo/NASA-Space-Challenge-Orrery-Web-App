## Getting Started

### Prerequisites
- Ensure you have **Node.js** and **npm** installed.

### Installation
  - npm init -y
  - npm install babylonjs-
  - npm install lite-server --save-dev
   
### Running the app
Move to src directory and execute
 - npx live-server
